# Tab Produits

<figure><img src="../../../.gitbook/assets/image (19).png" alt="" width="375"><figcaption></figcaption></figure>

## À quoi sert cette page ?

Voir **tous tes produits** (en ligne, brouillons).

Chercher, filtrer, **ajouter**, **importer** en masse ou **exporter**.

Vérifier vite : **stock**, **prix**, **bénéfice (“Gagner”)**, **vues**, **date**.

## Les gros boutons en haut

<figure><img src="../../../.gitbook/assets/image (20).png" alt="" width="316"><figcaption></figcaption></figure>

**Ajouter un nouveau produit** : créer un produit à la main.

**Import** : **ajouter beaucoup de produits d’un coup** depuis un fichier.

**Export** : télécharger la liste de tes produits dans un fichier (pour sauvegarde ou modifier en masse).

**Recherche** : taper un mot pour retrouver un produit.



## Les filtres (bandes grises)

<figure><img src="../../../.gitbook/assets/image (21).png" alt="" width="375"><figcaption></figcaption></figure>

Choisis **Dates**, **Catégorie**, **Type**, **Marque** puis clique **Appliquer** pour ne voir que ce qui t’intéresse.

**Actions en masse** : cocher plusieurs produits → appliquer une action (ex. mettre hors ligne).



## Le tableau (comment lire chaque colonne)

<figure><img src="../../../.gitbook/assets/image (22).png" alt=""><figcaption></figcaption></figure>

* **Image** : la photo principale.
* **Nom** : le titre cliquable du produit.
* **État** :
  * **En ligne** = visible par les clients.
  * **Brouillon** = pas encore visible.
* **Référence produit** (SKU) : ton code interne.
* **Stock** : _En stock_ / _Rupture de stock_.
* **Prix** : prix que voit le client.
* **Gagner** : ton **bénéfice estimé** (prix – coût/commission) si le calcul est activé sur la boutique.
* **Type** : icône qui indique le genre de produit (simple, avec variations, téléchargeable…).
* **Vues** : combien de fois la page a été vue.
* **Date** : dernière publication ou modification.
